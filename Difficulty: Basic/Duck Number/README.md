<h2><a href="https://www.geeksforgeeks.org/problems/zero-number2158/1">Duck Number</a></h2><h3>Difficulty Level : Difficulty: Basic</h3><hr><div class="problems_problem_content__Xm_eO" style="--text-color: var(--problem-text-color);"><p><span style="font-size: 18px;">A <strong>Duck</strong> number is a positive number which has zeroes present in it, For example, 3210, 8050896, 70709 are all Duck numbers. A number with only leading 0s is not considered as Duck Number. For example, numbers like 035 or 0012 are not considered as Duck Numbers.</span><br><span style="font-size: 18px;">The task is to check whether the given number <strong>num</strong> is a <strong>Duck</strong> number or not.</span></p>
<p><span style="font-size: 18px;"><strong>Examples:</strong></span></p>
<pre><span style="font-size: 18px;"><strong>Input:</strong> num = 707069
<strong>Output:</strong> YES
<strong>Explanation:</strong> 707069 contains a non-leading 0.</span></pre>
<pre><span style="font-size: 18px;"><strong>Input:</strong> num = 02364
<strong>Output:</strong> NO
<strong>Explanation:</strong> Contains leading 0.</span></pre>
<p><span style="font-size: 18px;"><strong>Constraints :</strong><br>1 ≤ Length of num ≤ 10<sup>4</sup></span></p></div><br><p><span style=font-size:18px><strong>Topic Tags : </strong><br><code>Strings</code>&nbsp;<code>Data Structures</code>&nbsp;
